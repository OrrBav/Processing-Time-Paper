# Processing-Time-Paper

This repository contains work-in-progress code for a research project investigating a distinct and stable sub-group of T-cell receptors (TCRs).

The project focuses on identifying and characterizing TCR sequences that persist across varying experimental conditions, with the goal of uncovering biologically meaningful and technically robust features of the immune repertoire.

⚠️ This repository is under active development and will be updated alongside the preparation of a manuscript.
